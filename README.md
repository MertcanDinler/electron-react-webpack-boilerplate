### Install

#### Clone this repo

```
git clone https://github.com/mrtcndnlr/electron-react-webpack-boilerplate.git
```

#### Install dependencies

```
npm install
```
or
```
yarn
```

### Usage

#### Run the app

```
npm run dev
```
or
```
yarn dev
```

#### Build the app (with electron)

```
npm run electron:build
```
or
```
yarn electron:build
```

#### Build the app (only react app / without electron)

```
npm run build
```
or
```
yarn build
```
